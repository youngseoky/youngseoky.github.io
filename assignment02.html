Question 1. plot(pressure, pch=16) # Can you change pch?

Yes, we can change the pch. If we change pch, the shape of the points on the graph changes. For example, if we change pch to 1, points appear as a hollow circle like the first one on the right in the figure below. Question 2. points(x, y1, pch=16, cex=3) # Try different cex value?

Changing the cex value changes the size of the point. That is, the larger the number, the larger the size of the point. For example, if you look below, the first picture is when the cex value of the black point is set to 1, and the second picture is when the cex value of the black point is set to 5.
