Assignment03

1. Run anscombe01.R (in class GitHub)

a. Compare the regression models

The figure shows that the 1st model is relatively suitable (linear relation) compared to the other models. It seems that the 2nd model is not suitable because the coefficient line is a straight line but the plots appear to follow a curvelinear pattern. The 3rd figure shows that the model suggested is not suitable because there is extreme value and this is affecting the coefficient line. Here, the 4th model is not suitable because there are only two x values of the observations, and all x values are at 8 except for one observation.
